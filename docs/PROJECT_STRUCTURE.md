# Project Structure

This document records the initial foundation layout for the Student Attendance Monitoring and Analytics System.

Feature modules should be added inside the existing client and server folders as the product grows.

```text
client/
  src/
    components/
    pages/
    layouts/
    contexts/
    hooks/
    services/
    utils/
    assets/

server/
  config/
  controllers/
  middleware/
  models/
  routes/
  services/
  database/
  uploads/
  utils/
```
